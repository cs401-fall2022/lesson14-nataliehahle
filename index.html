What is Typescript?

Overview

TypeScript adds additional syntax to JavaScript to support a tighter integration with the editor and allows easier error detection. Typescript offers all the features of Javascript and is adapted to work with existing Javascript code. All of our code can easily be implemented to use TypeScript.
